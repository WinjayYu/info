## 查询指定作者文章列表

**URL：** 
- ` /sendemail `
  
**请求方式：**
- POST

**备注：**
- 免登录

**参数：** 

|参数名|必选|类型|说明|默认值|
|:----    |:---|:----- |-----   |------  |
|email |是  |String |注册邮箱   |无 |

 **数据模型**

``` 
{
    "state": 请求状态,
    "info": 请求返回消息
}
```

 **返回示例**

``` 
{
  "state": true,
  "info": "获取验证码成功!"
}
```
