## 注册新用户

**URL：** 
- ` /register `
  
**请求方式：**
- POST

**备注：**
- 免登录
- 密码大于6位

**参数：** 

|参数名|必选|类型|说明|默认值|
|:----    |:---|:----- |-----   |------  |
|username |是  |String |用户名   |无 |
|password |是  |String |密码（至少6位密码）   |无 |
|email |是  |String |注册邮箱   |无 |
|verifyCode |是  |int |邮箱验证码   |无 |

 **数据模型**

``` 
{
    "state": 请求状态,
    "info": 请求返回消息
}
```

 **返回示例**

``` 
{
  "state": true,
  "info": "注册成功！"
}
```
