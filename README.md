## 前端安装

``` bash
# 进入前端代码目录
cd ./src/client

# 安装依赖
npm install

# 使用webpack打包build
npm run build
```

## 后端安装

``` bash
# 进入后端代码目录
cd ./src/server

# 安装依赖
npm install

# 运行
node app
```
