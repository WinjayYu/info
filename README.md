## 前端安装

``` bash
# 进入前端代码目录
cd ./src/client

# 安装依赖
npm install

# 使用webpack打包build
npm run build
```
访问 `src/client/dist/index.html`

## 后端安装

``` bash
# 进入后端代码目录
cd ./src/server

# 安装依赖
npm install

# 运行
node app
```
访问 `localhost:8081`
